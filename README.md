## Installation with conda (recommended)
```bash
conda create -n image_restoration python=3.8
conda activate image_restoration
pip install -r requirements.txt
```

## Download requisite checkpoints
```bash
bash download-weights
```

## Launch the demo
```
python main_gui.py
```
